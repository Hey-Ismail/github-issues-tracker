1️⃣ What is the difference between var, let, and const?

- `var` is the old way to make a variable.
- `let` is used when the value can change.
- `const` is used when the value we don't want to change.

2️⃣ What is the spread operator (...)?

- The spread operator takes out values from an array or object.
- It is often used to copy or combine .

3️⃣ What is the difference between map(), filter(), and forEach()?

- `map()` makes a new array by changing each item.
- `filter()` makes a new array with only the items that match the given condition.
- `forEach()` goes through each item to do something, but it does not return a new array.

4️⃣ What is an arrow function?

- An arrow function is a shorter way to write a function of ES6--js.
- Example: `const add = (a, b) => a + b;`

5️⃣ What are template literals?

- Template literals are strings written with backticks.
- They let you put variables inside a string with `${}`.
- Example: `` `Hello, ${name}` ``
